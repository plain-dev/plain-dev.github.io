苹果在macOS Mojave正式推出了深色模式，它能让我们在夜间使用电脑更加的舒适，但也有很多地方不够完善，比如以下几点

引言

切换不够方便，没有快捷开关

macOS的深色模式并不支持自动开启，也没有一个快捷方式，每次切换都需要到设置>>通用里去找，很麻烦，也很费时间。起初每次还都去手动设置一下，到最后就懒得调了，一直保持在深色模式，看来是我太懒了吧。

深色模式不会作用于网页内容

对于浏览器来说，比如Safari开启深色模式后只是对浏览器应用本身做了适配，但网页里的内容并没有变为深色，这对熬夜党来说，晚上看白色背景的网页简直是噩梦，如果你开灯当我没说(我个人到晚上是只开一个桌面台灯的，喜欢把屋顶的灯关闭。)

好在macOS良好的软件生态，以上我遇到的这些玩问题，都可以通过第三方的软件来解决，下面就介绍给大家，两种需求对应了两款小工具，一是解决开关问题的NightOwl，二是解决网页深色模式的Safari插件Dark Mode for Safari。

软件推荐

NightOwl

  • 介绍

NightOwl提供了状态栏快捷切换浅色和深色模式、定时开启、分应用开启等功能，基本满足了我的使用需求。

  • 使用

    • 快捷启动

      在通知栏点击NightOwl的LOGO，点击Light或Dark，就可以快捷切换浅色或深色模式，如图所示

    • 定时切换

      NightOwl可以根据自己的喜好，定时设置深色模式的开启，如图所示

    • 分应用开启

      NightOwl可以对单个引用设置开启和关闭，比如我对App Store开启了单独设置，在打开深色模式后App Store的外观并没有跟着改变。

  • 下载

这款软件可以再NightOwl官网免费下载

  • 小结

以上就是NightOwl的基本使用，这些功能让深色模式有了更大的用武之地，开心。

Dark Mode for Safari

  • 介绍

Dark Mode for Safari是一款Safari浏览器插件，可以让网页变成深色模式，在我使用几天后，效果还是很不错的，常用的网站也都有适配，比如哔哩哔哩、V2EX、知乎、简书、掘金等。

  • 使用

Dark Mode for Safari的使用很简单，它的使命就是要让网页变黑，下载插件后，在Safari的偏好设置>>扩展中将其打开就可以使用了。

Dark Mode for Safari的开关模式主要有以下几种

  1. Turn Off：保持关闭

  2. Turn On Now：保持打开

  3. When macOS Dark Mode is On：跟随系统深色模式⭐️(推荐用这个模式，跟随系统变化，很方便)

  4. Scheduled：定时开启，可通过设置开始或结束时间，开启或关闭网页深色模式

Dark Mode for Safari也提供了多种深色模式的选择(已Github为例)💻

  1. Dark Mode：默认的深色模式
  1. Softer Dark Mode：比Dark Mode颜色要淡一些
  1. Mono Mode：黑白单色模式
  • 下载

在Mac App Store搜索Dark Mode for Safari就可以下载到，售价12元。

  • 总价

Dark Mode for Safari解决了我对网页深色模式的需求，并且使用了几天下来感觉良好，对我来说这12元`花的是值得的✌️
